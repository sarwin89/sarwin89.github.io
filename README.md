# sarwin89.github.io

Personal portfolio site for Sarwin Chandran, designed as a multi-page "Discover Me" experience instead of a single generic landing page.

## Overview

This site brings together:

- Physics notes and modeling work
- Code systems and software builds
- Art and visual studies
- Captures and observation-driven image work
- Creatives, including poetry, lyrics, demos, and fragments
- Blog entries for longer reflections and project write-ups

The portfolio is fully static HTML/CSS/JS and is built to work well across desktop and mobile with light and dark themes.

## Current Page Structure

- `index.html` - homepage
- `physics.html` - physics notebook
- `codes.html` - code systems
- `artworks.html` - art archive
- `captures.html` - captures / observation work
- `creatives.html` - poetry, music, and cross-medium creative work
- `blog.html` - long-form writing

## Legacy Redirect Pages

These older routes are kept so existing links still land somewhere useful:

- `astrophotography.html` -> `captures.html`
- `poetry.html` -> `creatives.html`
- `music.html` -> `creatives.html`

## Tech

- Static HTML pages
- Shared styling in `css/custom-style.css`
- Shared interactions in `js/main.js`
- Google Fonts for typography

## Local Preview

Because this is a static site, you can preview it by opening `index.html` directly in a browser or by serving the repo with any simple static server.
