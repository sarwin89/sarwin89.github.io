Why this layout works better

The portfolio makes more sense when each kind of work gets its own page and the homepage focuses on navigation rather than compression.

That means Physics can stay analytical, Code can focus on tools and systems, Art can stay visual, Captures can stay observational, Creatives can mix poetry and music naturally, and Blog can keep the long-form thinking visible.

The result is simpler to browse and more consistent from page to page.

Ground rules
Different page, one palette
Each page gets its own color family and keeps it consistent across cards, frames, and accents.
Compact navigation
The sidebar stays short enough to fit without forcing a scroll-heavy experience.
Responsive first
Smaller phones now get tighter media sizing and less visual bulk.

How to browse it

If you want the academic route, try Physics -> Code -> Blog.

If you want the broader picture, move through Art, Captures, and Creatives afterward. That route shows how the technical and personal sides of the work keep informing each other.
