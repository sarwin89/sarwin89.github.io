Repeatable formats

A few strong post shapes make it easier to keep writing without lowering the quality.

Good formats reduce friction. They let the journal stay active while still giving each post enough structure to feel like a real entry.

Format

Build log

What changed, why it changed now, what improved, and what still feels unfinished.

Format

Weekly note

One thing that moved, one thing that stalled, one image worth keeping, and the next small step.

Format

Process essay

Start with a scene, show the middle, name the hard part, and end with the lesson or next move.

Why keep this public

The journal makes the archive more legible because it keeps the reasoning around the work visible.

Without the journal, the site would still show outcomes, but it would lose the thinking between them. That middle layer matters academically and personally.

It is also where the crossovers appear most clearly: code leaking into poems, drawings changing how I write, technical work learning better patience from image-making.

Working principle

The post should sound like I finally had enough room to explain what mattered, not like a caption stretched into paragraphs.
